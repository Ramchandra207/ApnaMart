# 🛒 Apna Mart

A modern MERN Stack Multi-Vendor E-Commerce Marketplace where customers can browse products, vendors can manage their stores, and admins can manage the complete platform.

## 🌐 Live Demo

### Customer Website
https://apna-mart-web.vercel.app

### Backend API
https://apnamart-api-fkpg.onrender.com


# Features

## Customer

- User Registration & Login
- JWT Authentication
- Browse Products
- Product Search
- Product Categories
- Vendor Stores
- Shopping Cart
- Order Placement
- Order History
- User Profile

## Vendor

- Vendor Login
- Dashboard
- Add Products
- Edit Products
- Delete Products
- Manage Orders
- Store Management

## Admin

- Admin Dashboard
- Manage Users
- Manage Vendors
- Manage Products
- Manage Orders
- Categories Management
- Coupons
- Reports
- Notifications

# Installation


git clone https://github.com/Ramchandra207/ApnaMart.git

cd ApnaMart/jiet-marketplace

npm install

npm run dev
